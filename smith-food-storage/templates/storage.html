My Storage

This is a place where you can see all of the food you added from Add Food page. There is a search bar to simplify finding an item to remove when it’s used up. You can also sort by expiration date so you’re always using up the soonest‐expiring items. Remember, the icons at the top let you jump between Home, Add Food, and My Storage.
